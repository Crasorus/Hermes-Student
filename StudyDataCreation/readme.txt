Study Data Creation
===================
Tools for generating and reviewing test data for Hermes workflows.

  index.html            Browser-based UI for creating synthetic study data
                        (RTSM, ERP, CTMS files) to populate data drops.

  data_drop_viewer.html Browser-based viewer for inspecting generated data
                        drop files before feeding them into Hermes.

Open either file directly in a web browser — no server required.
