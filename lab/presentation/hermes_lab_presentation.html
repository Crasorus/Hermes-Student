Hermes 1 · What is Hermes?

A team of seven AI agents that run your clinical supply chain.

Each agent is a specialist. They collaborate autonomously — sharing data, raising signals, and producing decisions. All seven are defined entirely in markdown. No code required.

Supervisor
Orchestrates workflows, validates study packages, routes work
Demand Analyst
Enrollment, dispensing, screen failures, consumption rates
Supply Analyst
Stock positions, safety stock, reorder points
Logistics Specialist
Shipments, cold chain, customs, approved vendors
Compliance Manager
GxP audit trails, regulatory compliance, deviations
Insights Analyst
Anomalies, trends, risk hotspots across all signals
Reporting Agent
Executive summaries, deviation reports, ops updates
All seven
Configured per study. Run on demand. Output a complete picture.

Hermes 3 · How is it built?

Five layers. 100% prompt-driven. No code.

Every behaviour in Hermes is defined in plain English in a markdown file you can open, read, and edit. No programming required to build, customise, or extend it.

Layer 5 Runner Orchestration engine that runs the workflows and calls Claude automatic
Layer 4 Workflows Sequences that define which agents run, in what order, with what routing 5 workflows
Layer 3 Agents AI specialists with defined identities, procedures, and skill assignments 7 agents
Layer 2 Skills Atomic capabilities — calculators, parsers, validators, summarisers 76 skills
Layer 1 Study Package Your study's config, supply network, policies, reference data, and CSV data drops per-study
Hermes 3 · How is it built? — The skills library

76 skills across 8 functional categories.

Skills are the building blocks. Agents pick the skills they need from a shared library — the same skill can serve multiple agents.

  • Data Ingestion (DI) — 12 skills · validate, parse, load study data
  • Demand Forecasting (DF) — 9 skills · consumption, enrollment, scenarios
  • Supply & Inventory (SI) — 10 skills · stock, reorder, expiry, gaps
  • Logistics & Trade (LT) — 8 skills · lead time, cold chain, customs
  • Compliance & Regulatory (CO) — 10 skills · GxP, label, deviation, SOP
  • Analytics & Insights (AI) — 9 skills · anomaly, risk, trend, KPI
  • Reporting & Comms (RC) — 10 skills · summaries, dashboards, alerts
  • Supervisor & Orchestration (SO) — 8 skills · routing, preflight, exceptions
