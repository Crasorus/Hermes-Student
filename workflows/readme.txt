Workflows
=========

workflows.json is the main definition document for how workflows will run !

### Human Readable

This folder also contains the five Hermes workflow definitions.
Each file describes one end-to-end workflow that the agent team can execute.

  WF-01  Demand Signal Refresh
  WF-02  Supply Plan Generation
  WF-03  Protocol Amendment Impact
  WF-04  Routine Monitoring
  WF-05  Supply Plan Execution

These are reference documents for human understanding.


REMEMBER: Runtime step definitions and task sequences live in workflows/workflows.json.
